HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Metadata sets criteria of other usable data ie characterset etc
  • Child elements are elements nested inside other elements, known as parent elements
  • The "src" attribute defines locations for images
  • The "alt" attribute is also important as it is used for users that use screen readers and for improving search engine rankings

CSS

  • There are three types of CSS: inline, internal, and external
  • External is the most flexible and most commonly used, but must be linked
  • CSS is used to style the content of the HTML file
  • class attrributes can be assigned to elements that we want to share certain styles

Git

  • git status: Checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git push: transmits the local branch to the repository branch where it can be reviewed and merged

JavaScript

  • Control flow is the order in which a computer executes code in a scripy
  • A Variable is a named container that allows us to store data in our code
  • An array is a collection of variables that are indexed at a certain value 0-x
  • Loops are used to perform tasks for an established amount of time
  • Functions allow tasks to performed when called
